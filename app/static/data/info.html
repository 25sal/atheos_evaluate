Tool Info

Il presente tool è concepito per l'esercitazione e l'autovalutazione dell'apprendimento di linguaggi di programmazione attraverso il test di esercizi proposti allo studente.
Per utilizare il tool lo studente deve:
  1. Selezionare il linguaggio di programmazione e cliccare il pulsante invio.
  2. Selezionare la traccia e visualizzarla attraverso il link preposto.
  3. Scaricare il template dell'esercizio e il file di input se presenti cliccando sul nome del file.
  4. Risolvere l'esercizio e caricare la soluzione attraverso l'apposito controllo.
  5. Cliccare sul tasto test per eseguire il test.
La pagina visualizza il tempo a disposizione per risolvere l'esercizio e il risultato dell'ultimo test eseguito.
Nel caso di una prova ufficiale prima dello scadere del tempo lo studente deve scegliere se consegnare attivando l'apposito controllo. La versione consegnata della soluzione verrà discussa in sede di esame. Per discutere la soluzione in sede di esame è sufficiente il superamento di almeno un test.
